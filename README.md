Analyzed customer transaction and interaction data to understand purchasing behavior and identify key patterns. Performed data cleaning and exploratory data analysis to uncover trends related to customer segments, purchase frequency, and spending behavior. Built interactive dashboards to visualize insights and support data-driven business decisions.

Key Analysis Areas:
Customer segmentation
Purchase frequency & recency analysis
Average order value (AOV)
Retention and repeat customer analysis

Tools Used:
Python (Pandas, NumPy),
SQL,,
Power BI / Excel,
Data Cleaning & EDA
